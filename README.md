# 项目简介
	项目vue2.0学习演示，前后端分离，后端为节点直接解析JSON文件进行数据处理，前端为VUE框架;

## 技术栈
	vue2 + vue-router + webpack + ES6 + less + axios
## 项目运行
git clone https://github.com/hbdxlc/imitate_elm.git

cd imitate_elm

cd cms

npm install

cd ../

cd sell

npm install

分别启动即可
